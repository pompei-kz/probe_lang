Появился новый термин: Контекстное меню выбора выражения (ContextMenuSelExpr). Это меню должно открываться на квадратике выражения с кубиком.
Это меню многоуровневое:

  Пункт[1]: "Этот" -> от него открывается второй уровень:
    Пункт[2]: "Объект" -> устанавливает unit_e.type = 'ThisObject'
    Пункт[2]: "Юнит"   -> устанавливает unit_e.type = 'ThisUnit'
    Пункт[2]: "Метод"  -> устанавливает unit_e.type = 'ThisMethod'
  Пункт[1]: "Юнит" -> Открывает форму выбора юнита.

(на первом уровне два пункта, в квадратных скобках - уровень, на котором появляеться данный пункт)

Форма выбора юнита состоит из списка юнитов.
  Список с подкачкой. У списка есть скрол.
  При прокручивании скрола идёт автоматическая докачка.
  Если больше данных нет, нужно написать тусклым текстом в круглых скобках: "Всё, данных больше нет".
  Над списком текстовое поле фильтра.
  В списке появляются те юниты, которые содержат слова из фильтра в последовательности как в фильтре.

  При клике на юнит в списке, происходит сохранение этого юнита так: unit_e.type = 'Unit', unit_e_unit.unit_id = ИД_выбранного_юнита
